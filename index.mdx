## What is Telebot?

Telebot is a bot framework for the [Telegram Bot API](https://core.telegram.org/bots/api) written in Go. Rather than providing a thin 1:1 wrapper over the API, Telebot focuses on developer ergonomics and performance — giving you a clean routing system, composable middleware, and transparent file handling out of the box.

```go go.mod
module gopkg.in/telebot.v4
```

```bash
go get -u gopkg.in/telebot.v4
```

## Minimal example

```go main.go
package main

import (
	"log"
	"os"
	"time"

	tele "gopkg.in/telebot.v4"
)

func main() {
	pref := tele.Settings{
		Token:  os.Getenv("TOKEN"),
		Poller: &tele.LongPoller{Timeout: 10 * time.Second},
	}

	b, err := tele.NewBot(pref)
	if err != nil {
		log.Fatal(err)
	}

	b.Handle("/hello", func(c tele.Context) error {
		return c.Send("Hello!")
	})

	b.Start()
}
```

## Key features

<CardGroup cols={2}>
  <Card title="Command routing" icon="route">
    Route `/commands`, text patterns, and all Telegram update types through a single `Handle()` API.
  </Card>
  <Card title="Middleware" icon="layer-group">
    Chain middleware globally, per-group, or per-handler. Includes Logger, AutoRespond, Whitelist, and Recover out of the box.
  </Card>
  <Card title="Transparent File API" icon="file">
    Send files from disk or URL. Telebot automatically uploads on first send and reuses the Telegram FileID on subsequent sends.
  </Card>
  <Card title="Keyboards" icon="keyboard">
    Build reply and inline keyboards with a fluent markup builder. Buttons act as routing endpoints.
  </Card>
  <Card title="Inline mode" icon="magnifying-glass">
    Handle inline queries and respond with typed result objects for photos, articles, GIFs, and more.
  </Card>
  <Card title="Layout system" icon="table-columns">
    Define bot configuration, keyboards, and locales in YAML. Switch user locale at runtime.
  </Card>
</CardGroup>
